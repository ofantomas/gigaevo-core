## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| YAML file -> manifest | Untrusted experiment.yaml parsed by Pydantic |
| Redis -> manifest lock | Redis used for distributed locking; requires network access |
| ps output -> PID list | Subprocess output parsed for process management |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | manifest.py _write_manifest_atomic | mitigate | Atomic write via tmp+rename; Redis lock prevents concurrent writes |
| T-05-02 | Denial of Service | manifest.py _acquire_lock | mitigate | Lock TTL prevents permanent lock; timeout on acquire |
| T-05-03 | Information Disclosure | flush_ops.py kill_workers | accept | Process kill is admin-only operation; no PII exposed |
</threat_model>
