```python
def resolve_plugin(manifest=None) -> type[WatchdogPlugin]:
    """Resolve plugin: explicit watchdog_plugin field > solo fallback.

    Per D-01: No heuristic guessing. Delete _TASK_HEURISTIC entirely.
    """
    if manifest is not None:
        explicit = getattr(manifest, "watchdog_plugin", None)
        if explicit:
            if explicit not in _REGISTRY:
                raise KeyError(
                    f"Watchdog plugin '{explicit}' not found in registry. "
                    f"Available: {sorted(_REGISTRY.keys())}"
                )
            _log.info(f"Resolved plugin from manifest field: {explicit}")
            return _REGISTRY[explicit]

    # Fallback to solo -- no heuristic (D-01)
    if "solo" not in _REGISTRY:
        raise KeyError("No 'solo' plugin registered as fallback.")
    _log.info("Resolved plugin: fallback to 'solo'")
    return _REGISTRY["solo"]
```

3. Update the module docstring to remove "task heuristic" references.
