```python
@staticmethod
def _compose_g_program(d_code: str, g_points_list: list[list[float]]) -> str:
```
This method takes D's full code (containing an `entrypoint()` that returns a callable `improve(pts)`) and G's point configuration as a nested list. It produces a standalone Python program:

The template renames D's `entrypoint` to `_d_entrypoint` using regex: `re.sub(r'^(def\s+)entrypoint(\s*\()', r'\1_d_entrypoint\2', d_code, count=1, flags=re.MULTILINE)`.

Then wraps it:
```python
WRAPPER_TEMPLATE = '''\
import numpy as np

# G's original point configuration
_G_POINTS = np.array({g_points_repr}, dtype=np.float64)

# --- D's code (entrypoint renamed to _d_entrypoint) ---
{d_code_with_renamed_entrypoint}

def entrypoint():
    """D's improvement of G's points, as a valid G program."""
    improve_fn = _d_entrypoint()
    improved = improve_fn(_G_POINTS.copy())
    return np.asarray(improved, dtype=np.float64)
'''
```

**New `__call__` method for hook compatibility:**

```python
async def __call__(self) -> None:
    await self.inject()
```

**Rewritten `inject()` method:**

1. Get top-1 D program from `d_provider.get_top_k(1, higher_is_better=True)`
2. Get a random G program from `g_provider.get_opponents(1)` (uses fitness-proportional sampling)
3. Execute G's code via `run_exec_runner` to get its point output (the (11,2) array)
4. Compose wrapper program using `_compose_g_program(d_code, g_points)`
5. Execute the composed wrapper in subprocess to verify it produces valid output
6. If composed execution fails or output equals original g_points, return None (no improvement)
7. If composed output is valid (11,2) ndarray AND differs from g_points, inject the composed program
8. If `dg_tracker` is provided, record the improvement pair via `dg_tracker.record_improvement(d_id, g_id, delta)` where delta is computed as the difference in quality metric

Use `run_exec_runner` from `gigaevo.programs.stages.python_executors.wrapper` for both subprocess executions with a timeout of 30s each.

The "differs" check is: `not np.array_equal(composed_points, g_points)`. This is a conservative proxy for delta > 0.
