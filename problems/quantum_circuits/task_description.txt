OBJECTIVE
Write a Python code that modify the known of a Boolean symmetric tensor by reducing its rank. You will evolve the code for finding the best solution for specific signature tensor decompostion.

 
TASK
On input you will be given the boolean Matrix which rows are factors corresponding the decompostition of signature tensor T: 
$$
T = \sum_{r=1}^{R} P_{r} \otimes P_{r} \otimes P_{r}
$$
Your task is to reach minimum decomposition from the initial decompostion using the modification of TODD algorithm.

TODD algorithm:
Idea is to find a transformation via 1 rank matrices addition:
P -> P + zy^T
For the given z there exists subspace of {y} which can be used in transformation not changing the signature tensor
(the algorithm implementing this search is implemented in ToddGenerator)

If we choose z == P[i] or z == (P[i] ^ P[j]) and find that y[i] == 1 or y[i] ^ y[j] == 1 than factors in decomposition will be doubled and can be destroyed.

This algorithm is able to reach global minimum but it higly depends on the order of application such transformations.

YOUR ROLE
You can change choice function/ordering of the operation, you can try transformation wich increase the rank and not only decrease it for better exploration and etc.
