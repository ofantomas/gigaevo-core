**TASK DEFINITION – D-DIMENSIONAL SPHERICAL CODE**

**Challenge:** Implement a Python function that generates a set of $n$ points on the surface of a unit hypersphere in $\mathbb{R}^d$ ($S^{d-1}$) such that the **maximum cosine similarity** (inner product) between any pair of distinct points is minimized.

**Specific Benchmark:** $n=600, d=11$.
**General Requirement:** The code should be valid for any natural number $n \ge 2$ and dimension $d \ge 2$, defaulting to $n=600$ and $d=11$.

**GEOMETRIC SPECIFICATIONS**
1.  **Points (Items):**
    *   A collection of $n$ points: $X = \{x_1, x_2, \dots, x_n\}$.
    *   Each point $x_i \in \mathbb{R}^d$.
2.  **Constraint Surface:**
    *   Unit $(d-1)$-sphere: $\|x_i\|_2 = 1$ for all $i=1 \dots n$.
    *   Points must lie strictly on the surface of the hypersphere.
3.  **Objective:**
    *   Minimize the maximum pairwise inner product.

**OBJECTIVE FUNCTION**
Minimize the coherence (maximum cosine similarity): $\mu(X)$.
$$ \mu(X) := \max_{1 \leq i < j \leq n} (x_i \cdot x_j) $$
*Goal:* Produce a spherical code with the smallest possible maximum inner product (i.e., points should be as spread out/orthogonal as possible).

**CONSTRAINTS**
1.  **Spherical Constraint:** Every returned point must satisfy $\|x_i\| \approx 1.0$ (within floating point tolerance).
2.  **Count:** Exactly $n$ points must be returned.
3.  **Dimensions:** Points must have exactly $d$ coordinates.
4.  **Real Values:** No `NaN` or `Inf` values allowed.
