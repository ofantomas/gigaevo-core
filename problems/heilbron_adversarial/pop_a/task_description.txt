TASK DEFINITION -- HEILBRONN TRIANGLE PROBLEM (ADVERSARIAL CONSTRUCTOR)

You are a CONSTRUCTOR (Generator) in a two-population adversarial co-evolution game, analogous to a GAN:
- YOU (Pop A, the "Generator"): produce point configurations
- YOUR OPPONENTS (Pop B, the "Discriminator"): evolve perturbation operators that take your configuration and try to improve it, proving it is suboptimal

THE GAME
Your opponents are OTHER EVOLVING PROGRAMS. They receive your (11,2) point array and return a modified version. If their version has higher min_area than yours, they score points and YOU lose fitness. Each generation, your opponents get smarter -- they evolve increasingly sophisticated improvement strategies. A configuration that resists simple perturbations today will be cracked by more powerful methods tomorrow.

YOUR FITNESS (two equally-weighted components):
  fitness = 0.5 * quality + 0.5 * resistance

  quality    = min(min_area / 0.0365, 1.0)    -- how good is your min_area?
  resistance = 1 - mean(improvement / 0.0365)  -- how much can opponents improve you?

A high-quality configuration that opponents easily improve scores LOWER than a slightly-worse configuration at a true local optimum. You need BOTH quality AND resistance.

WHAT WINS THIS GAME
- Configurations at genuine local optima of the min_area landscape -- where no perturbation of any point increases the minimum triangle area
- The global optimum (min_area ≈ 0.0365) is perfectly resistant: no improvement is possible
- Shallow local optima will be escaped by increasingly sophisticated opponents. Aim for DEEP local optima or the global one.

OBJECTIVE
Return 11 distinct 2D coordinates inside a unit-area equilateral triangle such that:
1) (Containment) All points lie inside or on the boundary of the enclosing triangle
2) (Distinctness) All points are distinct
3) (Non-degeneracy) No three points are collinear
4) (Maximality) The minimum area among all C(11,3) = 165 triangles is maximized
5) (Resistance) The configuration sits in a deep basin of the fitness landscape

Output: (11, 2) NumPy array  |  Target: min_triangle_area >= 0.0365

CONSTRAINTS
- Enclosing triangle: equilateral, area = 1.0, flat-bottomed
  Vertices: A=(0,0), B=(1.5197,0), C=(0.7598,1.3161)
- Exactly 11 distinct points, all within or on boundary
- No collinear triplets

HELPER FUNCTIONS (import from helper.py)
- `get_unit_triangle()` -> (A, B, C) vertices
- `get_smallest_triangle_area(coordinates)` -> float min_area
- `is_inside_triangle(points, a, b, c)` -> bool

OUTPUT FORMAT
Implement `def entrypoint():` that returns a (11, 2) NumPy array.
Use numpy, scipy, sklearn, or standard library as needed.
Fix random seeds if using randomness.
